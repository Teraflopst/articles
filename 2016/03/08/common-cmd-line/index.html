常用终端命令的含义

Mar 8, 2016

说明

这里记录的是 Mac 终端命令行的介绍，方便大家了解终端的常用操作、各个命令的全称、历史和发展，达到更好理解、熟悉运用这些命令的目的。

命令的附带参数这里不作详细介绍，可以使用手册页命令 man 查询各个命令的详情，如有需要请自行深入了解。这里的内容有 Cheat sheet 的意味。

维基百科 为主要参考资料。

忠告

永远不要执行你不清楚的命令！

快捷键

这里不会列出快捷键大全，因为你根本记不住多少个，经常使用和值得使用的命令才是好的命令。学习快捷键（或者其它）的正确方式是，你每次花费时间在做重复的事情并感到不耐烦，那么你最好花些时间精力了解和学习一下提高效率的方法，毕竟好心情是很重要的。

  • 光标左移：Ctrl + B
  • 光标右移：Ctrl + F
  • 光标至行首：Ctrl + A
  • 光标至行尾：Ctrl + E
  • 删除至行首：Ctrl + U
  • 删除至行尾：Ctrl + K
  • 删除至单词开始：Ctrl + W
  • 清屏：Ctrl + L
  • 插入最近删除：Ctrl + Y
  • 历史命令：↑ 或 ↓箭头，或 ⌃ + P、Ctrl + N
  • 搜索历史命令：Ctrl + R
  • 中止执行：Ctrl + C
  • 选中即复制，单击滚轮粘贴
  • 按住 ⌘，单击路径可以在 Finder 中打开

命令

$ man

作用：查阅某个命令的手册页，q 退出
英文：Manual pages

$ sudo

作用：通过安全的方式使用特殊的权限运行程序（通常为系统的超级用户）
英文：substitute user do 或 superuser do

$ cd

作用：用于改变工作目录
英文：change directory

$ mkdir

作用：新建目录
英文：make directories

$ pwd

作用：输出当前工作目录的绝对路径
英文：print working directory

$ ls

作用：列出文件
英文：list segment

$ nano

作用：把终端作为一个简单的文本编辑器
含义：一个文本编辑器
历史：nano 最早在1999年发布，名字叫 TIP。2000年1月10日，正式改名为nano。nano 这个名字来自于国际单位制词头 nano（10^-9 ，“纳”，如“纳米”），意思是 nano 是 pico（10^-12 ，“皮”，如“皮米”）的一千倍。

$ cat

作用：“Reads files sequentially, writing them to standard output.”
英文：catenate
参数：-n 对所有行显示行号

more

功能：类似 cat，以一页一页的显示方便使用者逐页阅读

$ less

功能：类似 more，
使用：退出 Q

$ which

作用：在环境变量 $PATH 设置的目录里查找符合条件的文件
使用：

who

作用：显示系统中有哪些使用者

参数：-r 若源文件是一个目录文件，此时将复制该目录下所有的子目录和文件

$ touch

作用：修改文件或者目录的时间属性，若文件不存在，新建文件
使用：

$ ls -l testfile  #使用ls命令查看文件的属性
