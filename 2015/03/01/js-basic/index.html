JavaScript 基本概念

Mar 1, 2015

变量

声明变量

命名规定：字母 、_ 或 $ 开头，后面可以包含 数字。
var 在函数外声明为全局变量，在函数内声明为局部变量。
let 语句块作用域，在如 if、for 等语句块中声明即可以变成局部变量。

变量提升

你可以引用稍后声明的变量，重点是变量的值不会被提升。
一个函数中所有var语句应尽可能地放在接近函数顶部的地方。

全局变量

实际上是全局对象的属性，全局对象是 window。
一般来说，在函数外用 var 声明的变量为全局变量，在函数内部声明的为局部变量。
值得注意的是，不是用 var 在函数内部声明的变量变成了全局变量。

常量

终于可以声明常量(constant)了，使用 const。
声明时必须被初始化为某个值。以后不可以通过赋值改变其值，也不可以在脚本运行时重新声明。
兼容性：Firefox 和 Chrome 很早就支持了，IE11 才支持。

数据类型

JavaScript语言可以识别下面 7 种不同类型的值：

  • 6 种是 原型 的数据类型:
    • Null. 一个表明 null 值的特殊关键字。
    • Undefined. 变量未定义时的属性。
    • Boolean. 布尔值，true 和 false.
    • Number. 表示数字，例如： 42 或者 3.14159。
    • String. 表示字符串，例如：”Howdy”
    • Symbol. (ECMAScript 6)。它的实例是唯一且不可改变的。
  • 以及 Object 对象
